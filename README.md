# วัตถุประสงค์
1.เพื่อช่วยเหลือนักศึกษาในการหาหอพักให้สะดวกและรวดเร็วมากขึ้น

2.เพื่อช่วยให้นักศึกษาได้ข้อมูลของหอพักอย่างครบถ้วน

3.เพื่อช่วยหอพักในการเพิ่มโอกาสการหาผู้เช่า
